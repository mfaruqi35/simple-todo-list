# To-Do List

A simple to-do list app built with vanilla HTML, CSS, and JavaScript.

---

## ✨ Features

- **Add task** — type and hit Add to create a new task
- **Complete task** — check the checkbox to mark a task as done
- **Edit task** — click the edit icon to modify an existing task
- **Delete task** — click the trash icon to remove a task
- **Progress tracker** — a circular progress ring showing how many tasks you've completed
- **Persistent storage** — tasks are saved in localStorage, so they survive a page refresh

---

## 🧰 Tech Stack

- **HTML**
- **CSS**
- **JavaScript** (Vanilla)

---

## 🚀 Getting Started

No installation needed. Just clone the repo and open `index.html` in your browser.

```bash
git clone https://github.com/mfaruqi35/todo-list.git
cd todo-list
```

Then open `index.html` in your browser.
